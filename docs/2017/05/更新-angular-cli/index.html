更新 Angular CLI

Sat May 20, 2017 by Jesse Chen

分成全域和專案套件

  1. 全域套件(Global package):

    npm uninstall -g @angular/cli
npm cache clean
npm install -g @angular/cli@latest

  2. 專案套件(Local project package):

    rm -rf node_modules dist
yarn add @angular/cli@latest --dev
