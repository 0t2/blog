Angular 4 環境準備

Sun May 7, 2017 by Jesse Chen

編輯器
  • Visual Studio Code
  • Bash on Windows
    • Node.js: 建議透過 nvm 安裝
    • Angular CLI: npm install -g @angular/cli
建立專案
ng new my-dream-app
cd my-dream-app
ng -v // 檢查 Angular 版本資訊
執行專案
ng serve

打開 http://localhost:4200/ 檢查執行結果
